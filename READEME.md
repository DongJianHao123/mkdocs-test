### MkDocs 
MkDocs是一个快速、简单、华丽的静态网站生成器，适用于构建项目文档。文档源文件以Markdown编写，并使用一个YAML文件来进行配置。

> 为了手动安装MkDocs，你需要在系统上安装Python，以及Python包管理器pip。详情见 <https://hellowac.github.io/mkdocs-docs-zh/>
#### 启动

1. 下载依赖
```
pip install -r requirements.txt
```
2. 运行
```
mkdocs serve
```
3. 打包
```
mkdocs build
```
4. 部署到github pages
```
mkdocs gh-deploy
```
